プロフィール


30歳でSEのKokiと申します。新卒でIT業界のベンチャー企業へ入社し、2年間法人営業に従事。その間は営業に 限らず、販促活動や大学生のマネジメントなど幅広い業務に挑戦。その後、営業よりもITエンジニアの方が、 人々の要望を叶えられ、自分の強みもより活かせると考えたことから、ITエンジニアに惹かれ、26歳の時に未経 験の状態でSEへ転職。そして、システムの保守からITエンジニアとしてのキャリアをスタート。しかし、大学生 の頃からフリーランスとして働いてみたいという気持ちがあったため、1度会社員を辞め、得意な英語を活かし てフリーランスの翻訳家に。そこからフリーランスとして1年半近くの経験を経て、会社員であることの恩恵を 改めて強く感じたため、会社員に戻ることを決断し、SEとしてのキャリアを歩み直し、現在に至ります。

私の特長やスキル

～ 特長 ～

1. 自走力

前職において、業務開始当初、業務の正確な手順書が存在せず、かつ前任者からの引き継ぎも全くありませんでした。しかし、様々な資料を読んだり、周りの方々にヒアリングをしたりと、自ら情報をかき集め、手順書を約1カ月で作り上げました。その上で、業務時間外で自発的にExcelについて学び、先輩社員の力を借りながら担当業務に関数やマクロを導入することで、作業工数を6割削減できました。

2. 根本から解決する力

物事について、仕組み・環境から見直して再構築し、問題を根本解決していくことが得意です。例えば、前職で担当した、お問い合せ対応の業務では、"そもそもなぜお問い合わせが発生するのか？"と根本から考え、分析し、仕組みを見直すことで、月5件以上発生していたお問い合わせ数を月0〜1件程度まで削減し、業務効率化に成功しました。

3. 向上心と貪欲な学習欲

向上心が強く、仕事においても常に現状に満足せず、"より短時間で、より質の高い成果を出したい"という気持ちが強いため、常日頃から業務に活かせそうな情報を絶えず学び続けています。例えばですが、最近でも、現職の業務と転職活動を並行して行いながらも、隙間時間を活用して2週間ほどでAWSの資格に合格することができました。

Djangoブログアプリ



フレームワークにDjangoを使用している、ブログのWebアプリです。簡単なアプリですが、CRUD機能・ページネーション機能・画像ファイルのアップロード機能など、Webアプリで使われることが多い機能がひと通り組み込まれております。今後は、テストコードの追加やAWSの導入などによって、さらに機能を充実させていきつつ、これらの技術を実務に活かしていければと思います。

使用言語など：HTML/CSS、Sqlite、Django、Python 3.11.4、Bootstrap
GitHub

画面の位置で変わる動物



マウスの位置に動物の顔が表示され、マウスを動かすと、その位置に応じて、表示される動物の顔が変わるアプリです。Processingのおかげで、少ない行数の簡単なコードを書くだけで、絵が描けてしまうので、プログラミングをより身近に、そしてその可能性を感じさせてくれるアプリです。

使用言語など：Python 3.11.4、Processing 4.2
GitHub

Youtube分析ツール



何かしらのキーワードを入力して検索し、表示されたYoutube上の動画を、さらにチャンネル登録者数でフィルタをかけて表示できる、Youtubeの分析ツールです。YouTube Data APIを用いており、フレームワークにはStreamlitを使用しております。さらに機能を充実させていき、Youtubeチャンネルを運営する人が、より高度な分析を行えるツールへと進化させていければと考えております。

使用言語など：Streamlit 1.23.1、Python 3.11.4
GitHub
